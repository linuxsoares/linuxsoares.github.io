My history with technology

Today I was seeing my linkedin and I see that I started with programing ten years ago! It’s makes me thinking about when started my curious with technology!

A long, long time ago, my father gave me a computer with the famous turtle logo. After some years I replaced this computer with Windows 95. Here start my curious with technology!

With this curious, I started trying to see what was behind it when I ran a Windows Explorer, but since it was an EXE file, I couldn’t see the code. Since I didn’t have internet access, nor much information, only Linux magazines from newsstands, I ended up giving up for now. Some time later, I ended up changing computers again, this time with an option that could use Linux. This was a bottomless pit, because I ended up buying a lot of magazines about Linux to learn how to use it.

Some versions of Linux that I used:

  • Kurumin
  • Conectiva
  • Debian
  • Ubuntu
  • Fedora

Among other options that I don’t remember at the moment.

But I always ended up blocked due to lack of knowledge, and I gave up again. Later on, some friends suggested I study Computer Science. So I started researching how I could afford to go to college since I was curious but lacked the money. So I decided I wanted to study, I just lacked the money, then someone, I don’t remember who, introduced me to the “Educate for Life” project. This project had partnerships with colleges, and these partnerships offered discounts of 60%, sometimes 80%. That’s when the curious boy with computers got into this “escort” thing.

And ever since I started my Computer Science course, I’ve only thought that this was my only chance to change my life, and it was!

Today I work with programming, and I’m still the curious kid I used to be, interested in languages and performance, and now seeing AI transforming my work, but making it more fun, because now I can create things much faster!
